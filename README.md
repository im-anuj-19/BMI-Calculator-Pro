# BMI Calculator Pro

A simple and user-friendly desktop application developed using " Python " and " Tkinter " to calculate Body Mass Index (BMI). The application provides BMI values, health categories, healthy weight range, and health advice based on user inputs.



# Features

- Calculate Body Mass Index (BMI)
- Supports multiple weight units
  - Kilograms (kg)
  - Pounds (lbs)
- Supports multiple height units
  - Meters (m)
  - Centimeters (cm)
  - Feet (ft)
- Displays BMI category
- Provides health advice
- Calculates healthy weight range
- Reset option
- Clean and easy-to-use graphical interface



# Technologies Used

- Python 3
- Tkinter
- ttk Widgets

# BMI Categories

 Below 18.5  - Underweight .
 18.5 – 24.9 - Normal Weight .
 25.0 – 29.9 - Overweight .
 30.0 and above - Obese .
